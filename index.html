About Me

Hi, I’m Chris. I’m an IT professional with 4+ years of experience in application support, web development, data analytics, and machine learning. I specialize in supporting enterprise applications, resolving production issues, optimizing performance, and ensuring smooth operations across Linux and Windows environments.

In my recent role, I provide end-to-end application support, from monitoring and troubleshooting to CI/CD deployments and API management using MuleSoft, Postman, and ReadyAPI. I also work on cloud infrastructure (Azure, AWS), manage release processes with Cherwell, and build monitoring dashboards with Dynatrace, Pingdom and ELK. Beyond support, I’ve developed native mobile apps for Android and iOS, worked on machine learning projects, and built automation and data solutions using Python, SQL, and Power BI. I’m passionate about building reliable systems and continuously improving how we support and scale technology.
